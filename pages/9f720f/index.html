    for...in和for...of区别

    # for…in

    是为遍历对象属性而构建的，不建议与数组一起使用

    # for…of

    在可迭代对象 (opens new window)（包括 Array (opens new window)，Map (opens new window)，Set (opens new window)，String (opens new window)，TypedArray (opens new window)，arguments (opens new window) 对象等等）上创建一个迭代循环，调用自定义迭代钩子，并为每个不同属性的值执行语句

    通过break (opens new window)关键字，提前终止循环。
