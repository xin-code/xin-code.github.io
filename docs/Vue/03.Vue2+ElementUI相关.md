## el-dialog 与外层均可操作

- :modal="false" 是否需要遮罩层

- :close-on-click-modal="false" 是否可以通过点击 modal 关闭 Dialog

- :star: 在CSS中添加如下样式

  ```css
  // 覆盖层元素增加可穿透点击事件
  .el-dialog__wrapper{
    pointer-events:none;
  }
  // 弹窗层元素不可穿透点击事件（不影响弹窗层元素的点击事件）
  .el-dialog{
     pointer-events:auto;
  }
  ```
