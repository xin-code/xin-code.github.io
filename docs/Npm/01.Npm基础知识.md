Npm基础知识点总结

<!-- more -->



## 基础操作

1. 查看版本号

   ```shell
   npm -v
   ```

2. 升级（最新）版本

   ```shell
   npm install npm -g
   ```

   

## 查看源

```shell
// 查看源
npm get registry
```



## 更换源

```shell
// 配置淘宝源
npm config set registry https://registry.npm.taobao.org

// 验证是否成功
npm config get registry

// 原始源
npm config set registry https://registry.npmjs.org
```



## 清除缓存

```shell
npm clean cache -force
```
