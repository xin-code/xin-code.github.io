## 自定义代码模板

- 文件-->首选项-->用户代码片段-->点击新建代码片段--取名\*\*.json 确定

- 以下以`Vue.json`为例

```json
{
  "Print to console": {
    "prefix": "<", // 用户使用给的时候 输入这个'<'的时候自动弹出提示模板
    "body": [
      "<template>",
      "  <div>$5</div>",
      "</template>",
      "",
      "<script>",
      "export default {",
      "  data() {",
      "    return {};",
      "  },",
      "  methods: {},",
      "}",
      "</script>",
      "<style lang='scss' scoped></style>"
    ],
    "description": "Log output to console"
  }
}
```
