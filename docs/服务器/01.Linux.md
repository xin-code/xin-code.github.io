### ChromeDriver

- 安装Chrome

  ```shell
  sudo dpkg -i google-chrome-stable_current_amd64_90.0.4430.72.deb 
  ```

- 安装Driver

  ```shell
  sudo mv chromedriver_linux64.zip /usr/local/bin/
  ```

- 查看版本

  ```shell
  google-chrome -version
  ```
